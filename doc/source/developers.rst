For Developers
==============

:class:`matcalo.core.base.MatCALO` provides the methods :attr:`infer` and :attr:`learn`
for triggering the *MatCALO* pipelines for learning and reasoning.

Reasoning
---------

    :Example:

    >>> from matcalo.core.base import MatCALO
    >>> calo = MatCALO()
    >>> calo.query = query
    >>> calo.threshold = threshold
    >>> calo.strategy = MatCALO.Strategy(strategy)
    >>> calo.infer()

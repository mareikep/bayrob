bayrob.utils.utils.pcc(X, fnames, ignorenans=True)#

Gets a matrix of data, where each row is a sample, each column a variable and returns two matrices containing the feature names and the pearson correlation coefficients of the respective variables

Parameters:

  • X – the input data matrix, each row represents a data point, each column a variable

  • fnames – the names of the variables, i.e. len(fnames) = len(X[i])

  • ignorenans – whether to ignore nan values in the data or not. Caution! When ignoring NaN values, the covariance matrix is built from only the available values

Returns:

two matrices: the feature-feature name and values for the respective PCC

bayrob.utils.utils.pnt2line_(pnt: List | Tuple, start: List | Tuple, end: List | Tuple) → Tuple#

Given a line with coordinates ‘start’ and ‘end’ and the coordinates of a point ‘pnt’ the proc returns the shortest distance from pnt to the line and the coordinates of the nearest point on the line.

Algorithm:

  1. Convert the line segment to a vector (‘line_vec’).

  2. Create a vector connecting start to pnt (‘pnt_vec’).

  3. Get the dot product of pnt_vec and line_vec (‘dot’).

  4. Find the squared length of the line vector (‘line_len’).

  5. If the line segment has length 0, terminate otherwise determine the projection distance from start/end.

  6. If t < 0, the nearest point would be on the extension closest to start, if t > 1, closest to end.

  7. Calculate the distance from pnt to the nearest point on the line segment (‘dist’).

Parameters:

  • pnt – Union(List, Tuple)

  • start – Union(List, Tuple)

  • end – Union(List, Tuple)

Returns:

Tuple

class Result(query_object, success=False, error=None, message=None, result=None)#

The result of either query or plan refinement in BayRoB. The object contains information about the success of the operation and provides information about possible errors or additional messages relevant to interpret the outcome.

property query_object: Search | Query#
property success: bool#

Returns True if the task was executed without errors, otherwise False.

property error: str#

Returns True if the task was executed without errors, otherwise False.

property message: str#

A message with additional information possibly relevant to interpret the result.

property result: Tuple[jpt.trees.JPT, jpt.variables.VariableMap] | List[bayrob.core.astar_jpt.State]#

The result of the task execution. Either a sequence of actions (plan refinement) or a tuple (cond, post) of a conditional tree and posterior distributions after reasoning over an individual model.

clear()#

Remove all information of the result except the passed query object.

adddatapath(path) → None#

Adds a path to an action model containing the model file containing an jpt.trees.JPT instance and a .parquet file containing the original data to train the model. After adding the path(s), the model is loaded. The model objects as well as the path to the datafiles are accessible using the attributes models and datasets.

Parameters:

path (str or iterable) – The path(s) to an action model containing data and model files

removedatapath(path) → None#

Removes one or more path(s) pointing to model and data files. The respective models are removed from the models.

Parameters:

path (str or iterable) – The path(s) to an action model containing data and model files

property query: Query | Search#

The query attribute is an instance of either Query or Search specifying the required information for reasoning over an individual model or refining a plan.

property datapaths: list#

The datapaths containing the paths action models.

Returns:

list of paths to the folders containing pickled jpt.trees.JPT models and the .parquet data file used for training the model

Return type:

List[str]

property models: Dict[str, jpt.trees.JPT]#

a mapping from model names to jpt.trees.JPT models

property datasets: Dict[str, str]#

search_astar() → None#

This function performs a plan refinement by searching a path from the initial state to the goal state defined in the bayrob.core.base.Search object passed to the class beforehand.

query_jpts() → None#

This function performs a query to an action model as specified in the bayrob.core.base.Query object passed to the class beforehand.
